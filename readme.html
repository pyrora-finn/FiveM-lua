FiveM Script

Dies ist ein einfaches Beispiel für ein FiveM-Skript, das die Benutzerregistrierung und -anmeldung ermöglicht.

Inhalt

  • Features
  • Installation
  • Nutzung
  • Lizenz

Features

  • Benutzerregistrierung und -anmeldung über das Spiel-Chat-System.
  • Speicherung von Benutzerdaten in einer SQLite-Datenbank.
  • Einfache Befehle zur Interaktion mit dem Skript.

Installation

1. Erstelle einen neuen Ordner in deinem resources-Verzeichnis deines FiveM-Servers.
   Benenne den Ordner my_fivem_script.

2. Füge die folgenden Dateien in diesen Ordner ein:
   - fxmanifest.lua
   - server.lua
   - client.lua

3. Öffne deine server.cfg und füge die folgende Zeile hinzu:
   start my_fivem_script

4. Starte deinen FiveM-Server.

Nutzung

1. Um einen neuen Benutzer zu registrieren, gib im Spiel-Chat den folgenden Befehl ein:
   /register  

2. Um dich anzumelden, verwende den Befehl:
   /login
